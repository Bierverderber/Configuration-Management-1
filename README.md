# Этап 1: 
Разработан минимальный прототип эмулятора терминала с графическим интерфейсом. Реализованы основные команды (ls, cd, exit), парсер переменных окружения и обработка ошибок. Заголовок окна динамически формируется из данных ОС пользователя.

# Этап 2:
Реализована настройка эмулятора через параметры командной строки. Добавлено логирование всех событий в XML-формат с сохранением имени пользователя. Поддержка стартового скрипта позволяет автоматически выполнять последовательность команд при запуске. Ошибочные строки в скрипте пропускаются с отображением сообщений об ошибках. Созданы тестовые скрипты для проверки всех параметров командной строки.
