# Этап 1: 
Разработан минимальный прототип эмулятора терминала с графическим интерфейсом. Реализованы основные команды (ls, cd, exit), парсер переменных окружения и обработка ошибок. Заголовок окна динамически формируется из данных ОС пользователя.

# Этап 2
Добавлена поддержка параметров командной строки для настройки путей VFS, лог-файлов и стартовых скриптов. Реализовано XML-логирование с учётом имени пользователя. Написаны тестовые скрипты для проверки работы эмулятора с различными параметрами.
